Complete each shape to fill the grid

Each patch contains exactly one clue: a number (how many cells in that patch) and a shape icon (or “any”). Inspired by LinkedIn’s Patches design notes and Shikaku-style rules.

  • Square — width equals height
  • Tall rectangle — taller than wide
  • Wide rectangle — wider than tall
  • Any of the above

Each clue shows filled cells / target area (e.g. 3/10). The icon further restricts valid dimensions when the patch is complete (except “any”). Invalid patches are cleared as soon as you finish a stroke.

Each patch covers at least two cells (no 1×1). Drag on empty cells to draw a new patch. To grow a partial patch, start the drag on any cell already in that patch and include empty cells in the rectangle. Tap once (no drag) on any cell of a patch to remove that whole patch. Hint removes every filled region that does not exactly match a correct solution patch (use Undo if needed).
